# Textures

Coloque aqui as texturas utilizadas nos elementos do ambiente SentinelVR.

## Texturas necessárias:

| Arquivo | Uso | Resolução recomendada |
|---|---|---|
| `monitor_frame.png` | Moldura metálica dos monitores | 1024x1024 |
| `control_panel.png` | Painel de controle da sala | 2048x2048 |
| `floor_tiles.png` | Textura do chão da sala de controle | 1024x1024 (tileável) |
| `wall_surface.png` | Textura das paredes | 1024x1024 (tileável) |
| `button_normal.png` | Botão em estado normal | 256x256 |
| `button_hover.png` | Botão em estado hover | 256x256 |
| `button_pressed.png` | Botão em estado pressionado | 256x256 |
| `sentinel_logo.png` | Logo SentinelVR para UI | 512x256 (com transparência) |

## Configuração no Unity:

- Texturas de ambiente: Texture Type = Default, sRGB = true, Generate Mipmaps = true
- Texturas de UI: Texture Type = Sprite, sRGB = true
- Normal maps: Texture Type = Normal Map
